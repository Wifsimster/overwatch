# Overwatch
Personal Home Automation for Raspberry Pi 3

Current functions:
- Date & time
- ESP8266 sensors
- Freebox 
- Netatmo
- Xiaomi Camera

Powered with :
- Vue.js 2.x
- Webpack
- MQTT
- Socket.io
- SQLITE
- RTSP stream
